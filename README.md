
#Aplication 
* Front-end developer

#### Instructions:


* Descargar el repositorio
* ```$ git clone https://github.com/JoseSulbaran/React-Redux-Cart.git```
* 
* Accede al directorio
* ```$ cd React-Redux-cart```
*
*Ejecutamos el siguiente comando. 
* ```npm install```
* ```npm start```
*
* Visit ```localhost:3000```
